# SwimFlow Analytics

Professional swimming training analytics platform. Track your performance, calculate SWOLF, and visualize your progress with interactive charts.

# Features
- **Data Visualization:** Interactive charts showing your training history.
- **Smart Metrics:** Automatic calculation of Power (Watts) and SWOLF score.
- **Modern UI:** Clean, responsive design with Dark Mode support.
- **Full Stack:** Built using the MERN stack.

# Tech Stack
- **Frontend:** React.js, Chart.js, CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas

# Installation
1. Clone the repository: `git clone https://github.com/ytuqete/swimflow-analytics.git`
2. Install dependencies for both `client` and `server`: `npm install`
3. Run the project: `npm run dev`
